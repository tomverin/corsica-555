# Corsica 555 — Checklist Perso Tom

Construite à partir de:

- `checklist.md` réglementaire BikingMan / Corsica 555.
- Checklist BRM400 personnelle.
- Retex BRM400: seul oubli identifié = **gilet de visibilité**.
- Choix course actuels: pas de prolongateurs, potence inversée, Race King 2.2, priorité confort / sécurité / épaule.

Setup portage prévu: sacoche top tube + Tailfin Speedpack. Petite sacoche de selle possible pour le petit outillage, mais les chambres VTT n'y tiennent pas.

Objectif: savoir quoi porter, quoi monter sur le vélo, quoi mettre dans chaque zone de portage, et quoi vérifier au départ.

## Nutrition À Emporter

Principe Corsica: glucides principaux portés depuis le départ, commerces = eau / salé / moral.

- [ ] Plan nutrition écrit par blocs.
- [ ] Objectif glucides course défini: ~32 h de fuel porté, pas 34 h pleines.
- [ ] Setup bidons: 1 bidon eau pure en permanence + 1 bidon glucides.
- [ ] Gels Nduranz: 16 unités, soit 1 gel toutes les ~2 h.
- [ ] Barres pour premières heures.
- [ ] Stock ravito nuit.
- [ ] Poudre: 32 doses de ~37-40 g glucides par bidon.
- [ ] Packing nutrition: 8 blocs de 4 h = 2 gels + 4 doses poudre par bloc.
- [ ] Électrolytes / sel sodium-potassium.
- [ ] Sel régime: 1.1 g = ~289 mg sodium + ~190 mg potassium.
- [ ] Dose chaude de référence: ~1.1 g sel régime dans le bidon eau quand besoin sodium.
- [ ] 2-3 aliments faciles d'accès.
- [ ] 2-3 options salées anti-sucré.
- [ ] Coca / caféine secours si décidé J-1.
- [ ] Plan points d'eau chargé / noté.
- [ ] Rappel règle: manger avant chaque secteur gravel / piste.
- [ ] Rappel règle: remplir si bidon <60 % à une fontaine.

## Contrôles Mentaux En Course

- [ ] Manger avant km 17, 33, 136, 253, 313, 389, 417, 433.
- [ ] Reset complet au km 433 avant Sequence C.
- [ ] Changer de position toutes les 20-30 min.
- [ ] Si épaule >2/10: ralentir, relâcher, mobilité courte, pas de passage en force.
- [ ] Si genou >2/10: baisser intensité, cadence souple, marcher si nécessaire.
- [ ] Si lucidité baisse: micro-rest 5-15 min avant de continuer.
- [ ] Si animal / patou: ralentir, s'arrêter, passer à pied si nécessaire.
- [ ] Si abandon / souci médical: appeler PC course avant toute autre décision.
