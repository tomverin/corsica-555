## Tactique terrain (cf. `terrain_analysis.md` — vfinale GPX)

### Trois séquences structurantes

- [ ] **Sequence A — km 12-42 (h+1 à +3)** : early gravel chain. Track suspect 17-25 (+4.3 %), puis gravel confirmé 33-42 (+4.2 %, ~+405 m, sommet ~978 m). Manger AVANT km 17. Mode warm-up, **pas test**.
- [ ] **Sequence B — km 137-186 (h+11 à +15)** : approche CP1 + point haut. Climb suspect 137-147 (+329 m vers ~1 275 m), descente -4.9 %, **CP1 Ghisoni km 164 (alt 658 m)** (refuel impératif), climb paved 164-175 vers **1 291 m**, descente -6.3 % paved → **transition rough km 178-184** (5 km à -6 %), continuation paved. Couches au km 175. Anticiper le changement de surface 178-184.
- [ ] **Sequence C — km 440-462 (h+26 à +29) — CLIMAX** : montée gravel +8 % vers 955 m + descente gravel **-8.9 %** sur 7 km, puis descente paved -4.5 %. Reset complet au km 433 (manger, hydrater, épaule). **Speed cap explicite** sur la descente. Marcher des sous-sections est tactique valable. Aucune compromise sur la sécurité ici.

### Secteurs spécifiques

- [ ] **CP1 Ghisoni km 164 (alt 658 m)** — feeding window obligatoire avant les 10.4 km à +6.1 % vers 1 291 m
- [ ] **CP2 Crocicchia km 333** — refuel avant longue descente -5.4 % km 343-352 vers ~150 m
- [ ] **km 99-109** — descente paved -6.5 % sur 10 km : un des deux vrais "free time windows" (drops OK)
- [ ] **km 195-209** — descente paved -5.0 % sur 14 km : le plus long descent de la course, paved (drops OK)
- [ ] **km 253-256** — dirt descent -4.4 %, hoods, freiner avant les courbes
- [ ] **km 313-316** — short steep rough descent (-6.3 %) : speed cap
- [ ] **km 389-411** — late mixed track corridor ~13.5 km : reset avant d'entrer
- [ ] **km 417-419** — descente gravel **-8.7 %** sur 2 km : single steepest rough descent. Marcher si doute.

### Position selon contexte

- [ ] **Cocottes** par défaut (90 % du parcours)
- [ ] **Drops** uniquement sur descentes paved techniques (km 99-109, km 195-209, km 174-178 hors transition)
- [ ] **Tops/cintre haut** sur les longues montées (164-175 vers 1 291 m, 240-252 vers 1 066 m)
- [ ] **Hoods uniquement** sur tout secteur off-road, jamais drops
- [ ] **Rotation toutes les 15-20 min** sur tarmac pour préserver épaule/nuque
- [ ] Off-road : regarder loin, freiner AVANT les courbes, jamais dedans

## Suivi corporel (règles douleur)

- [ ] **Genou** : si douleur > 2/10 → ralentir, marcher au besoin, anti-inflammatoire
- [ ] **Épaule** : si douleur > 2/10 → reset position, étirements courts, baisser pression posturale
- [ ] **Nuque** : tension persistante → arrêt court, mobilité cervicale
- [ ] **Mains** : changer de position fréquemment, prévenir engourdissement
- [ ] À chaque arrêt long / réveil bivouac : check 4 zones (genou / épaule / nuque / lombaires)
- [ ] Hydratation agressive AVANT les longues montées
- [ ] Sel / électrolytes toutes les 60-90 min sur effort

## Note de cadrage

L'objectif déclaré est **plaisir + capacité à se sentir bien sur tous les
secteurs**, pas un placement. La checklist hiérarchise donc :

1. Sécurité et conformité réglementaire (rien à débattre).
2. Préservation épaule / nuque / genou pour Corsica + Ascend Xtri (juillet)
   + TPR (septembre).
3. Capacité technique sur secteurs gravel (Race King + outillage).
4. Confort et nutrition pour 30+ h.
5. Optimisation chrono — en dernier.

Toute décision en course qui crée un conflit entre ces priorités tranche
dans cet ordre.
